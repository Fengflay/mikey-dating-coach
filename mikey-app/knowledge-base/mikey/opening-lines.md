# Opening Lines - First Message Guide

## Principle
The first message must be memorable. Avoid generic greetings like "hi", "hey", "how are you". Instead, create curiosity or emotional resonance.

## Good Openers by Context

### Dating App Match
- Reference something specific from her profile
- Playful observation > boring question
- Example: "I see you like hiking - but real question: sunrise or sunset hike person?"

### Mutual Friends Introduction
- Mention the connection naturally
- Don't over-explain how you know each other
- Example: "So [friend] tells me you're the one who makes the best [dish]. I'm skeptical."

### Cold Approach Follow-up
- Reference the specific moment you met
- Keep it brief - she may not remember
- Example: "The girl at [place] with the [distinctive detail]. This is [your name]."

## Common Mistakes
1. Too long - first message should be 1-2 sentences max
2. Too eager - don't compliment her looks immediately
3. Too generic - "hi how are you" gets ignored
4. Too aggressive - don't ask for a date in the first message
